Regions
Annotate genomic regions.
Installation
  • Make a copy of the regions folder (this folder) on your Desktop or other personal folder before running the application.
  • This application requires Java 1.6 or higher to run.
Getting Started
If you are running Windows XP/7/8, double click regions.bat to start.
If you are running Mac OSX, double click regions.command to start.
If you are running Linux, run regions.sh to start.
File formats
Regions will accept Excel xlsx, text, and BED files as input.
Excel and text files must have a header line and the first column must contain genomic coordinates of the form chrX:A-B (e.g. chr1:20000-20100).
BED files do not contain a header. The first column must contain chromosomes of the form chrX, the second column must contain start positions, the third column must contain end positions, and the fourth column must contain an identifier for the genomic location.
Data sources
